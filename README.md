## api

| 属性           | 类型          | 说明                           | 默认    |
| -------------- | ------------- | ------------------------------ | ------- |
| value(v-model) | array[string] | 输入的车牌号                   | []      |
| isMask         | boolean       | 是否展示遮罩层                 | true    |
| licenceLength  | Number        | 车牌号的长度（默认包含新能源） | 8       |
| slot           | slot          | 车牌号展示的内容               | default |
